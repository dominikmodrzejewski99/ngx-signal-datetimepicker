# Security Policy

## Reporting a Vulnerability

If you believe you've found a security issue in ngx-signal-datetimepicker, **please do not open a public GitHub issue**.

Instead, email **dominikmodrzejewski@wp.pl** with:

- A clear description of the issue
- Steps to reproduce / a proof-of-concept
- The library version and Angular version

You'll get an acknowledgement within 72 hours. We aim to ship a fix within 14 days for high-severity issues and publish a coordinated disclosure once a patched version is on npm.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

Once we reach 1.0.0, only the latest major will receive security fixes.
